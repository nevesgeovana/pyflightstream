# deprecated

Discontinued public items live here, grouped in one place so the
repository top level stays clean (author's decision, 2026-07-23).

Rules:

- An item moves here when it stops being maintained but is worth
  keeping visible in the repository (otherwise it leaves Git and goes
  to the author's private archive).
- Each item arrives with a dated note stating what replaced it.
- Nothing here is imported by `src/` or rendered into the docs; code
  deprecations inside the package (such as the `pyflightstream.files`
  shim) follow the DeprecationWarning path instead, per the SRS.
